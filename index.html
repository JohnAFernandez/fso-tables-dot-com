What is Ganymede


Ganymede is a database for table options used in the Freespace Open Engine, the open source continuation of Freespace 2, a video game released in 1999. It is named after the GTI Ganymede, which is the only retail ship used specifically for construction.

Freespace Open has been continuously developed since the source code's release and the number of table options has exploded. To facillitate search, maintenance and use of these table options, this database and website were created.

An API is available for use as well, documentation to be written.



Hard Light Productions


Hard Light Productions is the community that maintains Freespace Open and many of the modifcations available on the Knossos Launcher, as well as the various tools that help us create and maintain those mods. We strive to be a helpful, open, cooperative community.
